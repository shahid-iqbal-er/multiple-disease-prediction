Result History (Temporarily Disabled)

The result logging and history feature uses a MySQL database to store past predictions. In this local version of the project, database connectivity has been intentionally disabled.

You can still run predictions for Heart, Liver, Kidney, and Cancer using the navigation above. Each prediction is computed in real time using the trained machine learning models, but the results are not saved persistently.

In a production deployment, this page can be re-enabled to show detailed tables of past predictions once the database is configured.
